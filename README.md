# Olist Store Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes the Olist E-commerce dataset to provide business insights into sales, customers, products, shipping, and payment performance.

---

## Tools Used

- Power BI
- SQL
- Power Query
- DAX
- Excel

## Business KPIs

- Total Orders
- Total Sellers
- Total Products
- Payment Value
- Shipping Days
- YoY Growth
- Customer Reviews

---

## Key Insights

- Analyzed 99K+ orders.
- Credit Card is the preferred payment method.
- São Paulo has the highest number of orders.
- Faster shipping leads to better customer reviews.
- Business showed consistent Year-over-Year growth.
